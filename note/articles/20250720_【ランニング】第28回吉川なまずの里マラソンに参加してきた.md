# 【ランニング】第28回吉川なまずの里マラソンに参加してきた

【ランニング】第28回吉川なまずの里マラソンに参加してきた
=============================

[![くろもか / kuromoka](https://assets.st-note.com/production/uploads/images/14790436/profile_571ca71ab069abab3a69d7be84c82311.png?width=60)](/kuromoka)

[くろもか / kuromoka](/kuromoka)

2025年7月20日 11:33 フォローしました

3月30日に[第28回吉川なまずの里マラソン](https://runnet.jp/entry/runtes/user/pc/competitionDetailAction.do?raceId=369933&div=1)に参加しました！

2週間前の湯渡し100の疲れでマラソン向けの練習があまりできていない状態でしたが、頑張ってきました。

会場へは吉川美南駅からシャトルバスで移動

荷物預かりは300円払ってロッカーキーを借りて、ロッカーに預けるという珍しいシステムでした。

[![画像](https://assets.st-note.com/img/1765087035-qp5mpz2Ze0.jpeg?width=1200)](https://assets.st-note.com/img/1765087035-qp5mpz2Ze0.jpeg?width=2000&height=2000&fit=bounds&quality=85)

天候は曇り、気温は10度くらい、風もそこまでは強くなくコンディションに恵まれました。

[![画像](https://assets.st-note.com/img/1765087036-rSCejiCU2U.jpeg?width=1200)](https://assets.st-note.com/img/1765087036-rSCejiCU2U.jpeg?width=2000&height=2000&fit=bounds&quality=85)

小さい大会なので特に混雑もなくスタート

曲がり角が多いのが少し気になりましたが、コースは聞いていた通りほぼ平坦でした。途中サクラ🌸も見れました。

ハーフの記録も更新できました！

ゴール後は屋台もいろいろ出ていましたが、自販機のコーヒーで1杯やって帰りました。本当はコーラ飲みたかったけどやっぱ売り切れてた😂

時期的にはシーズン終わりの人も多そうな時期ですが、記録狙えるコースなので近場の人にはおすすめの大会です。
