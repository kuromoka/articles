# 記事執筆ガイドライン

このドキュメントは、このリポジトリで新しい記事を作成したり既存の記事を編集したりする際の、文体（トーン＆マナー）やフォーマットの指針をまとめたものです。

## 1. 基本的なトーンと文体

*   **丁寧で親しみやすい口調**: 語尾は「です・ます」調を基本とします。
*   **等身大の視点**: 完璧な自分を見せるのではなく、失敗や悩み、正直な感想を交えた「等身大の個人の記録」を大切にします。
*   **適度な絵文字の使用**: 文末などに 🙏, 🏮, 😂, 🥵, 😴 などの絵文字を使い、硬すぎない印象を与えます。
*   **謙虚さと率直さ**: 「当時のバカすぎる自分を殴りたい」「正直なんの警告か分からないこともありました」といった、自分に対する客観的で率直なツッコミを入れます。
*   **平易な表現**:
    *   **難しい言葉を避ける**: 専門用語や難しい漢字・熟語は極力避け、中学生でも理解できるような馴染みのある言葉を使います。
    *   **写実的・小説的な表現を避ける**: 過度に情緒的な描写や、比喩を多用した小説のような凝った言い回し、重苦しい写実的な表現は避けます。あくまで「ブログらしい軽やかさ」を優先します。

## 2. 記事の構成

1.  **タイトル**: 具体的な内容がわかるようにします（例：`【イベント名】〜してきた話`、`〜に入院することになった話`）。
2.  **導入**: 「どうも、kuromokaです」といった挨拶や、近況報告、その記事を書くきっかけなどを記述します。
3.  **目次**: `目次` という見出しの下に、セクションへのリンクをリスト形式で配置します。
4.  **本文**:
    *   時系列に沿った記述（「当日の記録」など）。
    *   写真（Markdown画像リンク）を多用し、その状況を説明する構成にします。
    *   重要なポイントや小見出し（`###`, `####`）で整理します。
5.  **持ち物・機材紹介**: レースや運転などの活動記録の場合、使用したアイテムを商品リンク付きでレビューするセクションを設けます。
6.  **感想・まとめ**: 最終的な達成感や、今後の目標（「次は20時間切りたい」「引き続き特訓していきます」など）を述べて締めくくります。

## 3. フォーマット・共通表現

*   **Markdownの使用**:
    *   強調は `**太字**` を使用。
    *   箇条書きリストを多用して読みやすくします。
*   **引用・リンク**:
    *   参考にした記事や商品リンクは、カード形式（タイトル、URL、価格、説明）を意識したフォーマットを模倣します。
*   **よく使う表現**:
    *   「めちゃくちゃ過酷でした」
    *   「正直〜だと思いました」
    *   「〜というモチベーションに歩いていました」
    *   「〜って感じでした」

## 4. コンテンツ別の注意点

*   **ランニング/ウォーキング**:
    *   キロ何分（例：`1km/10分`）といった具体的な数値を交えます。
    *   「エイドで何を食べたか」「どの地点で足が痛くなったか」などの臨場感を記述します。
*   **個人的なエッセイ/体験記**:
    *   その時の感情や考えの変化（「迷ったけど〜することにした」）を丁寧に描写します。
